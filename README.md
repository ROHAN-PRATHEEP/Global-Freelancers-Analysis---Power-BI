# Global Freelancers Analysis - Power BI Dashboard

Simple overview of the project purpose:  
This project analyzes global freelancer performance, skill distribution, pricing, activity status, and client satisfaction using Power BI.

---

## Description

This Power BI project provides a detailed analysis of freelancers across different countries and skills. The dashboard helps understand freelancer availability, pricing trends, project completion performance, and overall client satisfaction. It enables stakeholders to make informed decisions related to workforce planning, pricing strategy, and high-demand skills.

---

## Getting Started

### Dependencies

To run this project, you need the following:

- Windows 10 or later  
- Power BI Desktop (Latest Version)  
- Basic understanding of data visualization and Power BI
